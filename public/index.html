builder.js

This only works in Firefox.

Firefox allows you to implement a __noSuchMethod__ function for intercepting calls to un-implemented methods of an object.

Example 1: bold tag

builder.root("blinktag").b("hello");

Example 2: nested elements with an unordered list

builder.root("unordered").ul(function(b) { b.li("First").li("Second") } );
